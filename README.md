# Feedback_Amigo
En el ámbito escolar, los estudiantes no cuentan con un mecanismo estructurado para brindar retroalimentación a sus compañeros de manera **anónima, respetuosa y constructiva**. Esto genera:
- Falta de información sobre fortalezas y áreas de mejora entre los alumnos.
- Pocas oportunidades de mejorar habilidades interpersonales y trabajo en equipo.
- Dificultad para que docentes o coordinadores tengan un panorama del clima de grupo.

Actualmente, las evaluaciones entre pares suelen ser informales, dispersas y poco confiables. Por ello, se propone desarrollar una **plataforma web** que permita a los estudiantes calificar y dejar comentarios sobre sus compañeros, fomentando retroalimentación constructiva, de manera anónima (opcional) y segura.
